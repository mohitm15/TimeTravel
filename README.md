## TimeTravel

Time-travel is an application from which anyone can check the time of major Asian Cities. Currently it supports the following cities:

- Almaty  
- Amman   
- Ashgabat
- Baghdad  
- Baku   
- Bangkok  
- Beirut  
- Bishkek  
- Brunei  
- Damascus
- Dhaka   
- Dili   
- Dushanbe 
- Jakarta  
- Jerusalem
- Kabul   
- Kathmandu 
- Kolkata
- Kuala_Lumpur 
- Pyongyang 
- Manila  
- Riyadh  
- Sanghai  
- Seoul   
- Singapore 
- Taipei  
- Tashkent 
- Tbilisi  
- Tehran  
- Thimphu  
- Tokyo   
- Ulaanbaatar 
- Yerevan
